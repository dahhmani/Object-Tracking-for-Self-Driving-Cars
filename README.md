## Setup
```
git clone --recursive https://github.com/dahhmani/Object-Tracking-for-Self-Driving-Cars
cd <path to repository>
mkdir build
cd build
cmake ..
make
Run tests: ./test/cpp-test
Run program: ./app/shell-app
```
