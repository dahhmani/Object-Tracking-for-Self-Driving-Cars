\subsection*{Build}

Run the following commands in a new terminal\+: 
\begin{DoxyCode}
git clone --recursive https://github.com/dahhmani/Object-Tracking-for-Self-Driving-Cars
cd <path to repository>/scripts/
sh download\_pretrained\_model.sh
sh build.sh
\end{DoxyCode}
